Panduan Lengkap HTML dan CSS untuk Pemula: Mulai Membuat Website Sendiri

Pendahuluan

HTML (HyperText Markup Language) dan CSS (Cascading Style Sheets) adalah fondasi dari semua website. Dalam artikel ini, Anda akan mempelajari cara menggunakan HTML untuk menstrukturkan konten dan CSS untuk mengatur tampilan website.


Apa itu HTML?

HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web. Setiap elemen dalam HTML memiliki tag yang mendefinisikan jenis kontennya, seperti teks, gambar, atau tautan.

  • Tag dan Elemen HTML : HTML menggunakan tag untuk membungkus konten, seperti <h1> untuk judul, <p> untuk paragraf, dan <img> untuk gambar.
  • Struktur Dasar HTML:
<!DOCTYPE html> <html> <head> <title>Contoh Halaman</title> </head> <body> <h1>Halo, Dunia!</h1> <p> Ini adalah contoh halaman web pertama saya. </p> </body> </html>

Apa itu CSS?

CSS adalah bahasa styling yang digunakan untuk mengatur tampilan elemen HTML. Dengan CSS, Anda bisa mengubah warna, font, tata letak, dan masih banyak lagi.

  • Selektor CSS: Selektor digunakan untuk memilih elemen yang ingin Anda ubah tampilannya, seperti .class, #id, dan elemen langsung seperti h1, p, atau div.
  • Sintaks Dasar CSS:
body { font-family: Arial, sans-serif; background-color: #f0f0f0; } h1 { color: #333; }

Cara Menghubungkan HTML dan CSS

Ada tiga cara untuk menghubungkan CSS ke HTML:

  1. Inline CSS: CSS ditulis langsung di dalam tag HTML. Contoh :
    2. <p style="color:blue;">Teks biru ini adalah contoh inline CSS.</p>
  2. Internal CSS: Ditulis di dalam tag <style> di bagian <head>.
  3. External CSS: Ditulis di file terpisah dengan ekstensi .css, kemudian dihubungkan menggunakan tag <link>.

Cara Membuat Layout dengan CSS

Untuk membuat layout yang responsif dan terstruktur, kita bisa menggunakan beberapa teknik CSS berikut:

  • Flexbox: Digunakan untuk mengatur elemen dalam satu dimensi.
  • Grid Layout: Lebih cocok untuk mengatur layout dalam dua dimensi.

Contoh Layout Flexbox:

.container { display: flex; flex-direction: row; gap: 10px; }

Contoh Proyek Sederhana

Sebagai latihan, buatlah halaman landing sederhana dengan HTML dan CSS. Tambahkan header, konten utama, dan footer, serta atur tata letaknya menggunakan Flexbox atau Grid.

Template HTML Dasar untuk Latihan:

<!DOCTYPE html> <html> <head> <title>Landing Page Sederhana</title> <link rel="stylesheet" href="style.css"> </head> <body> <header> <h1>Selamat Datang di Website Saya</h1> </header> <main> <section> <h2>Tentang Kami</h2> <p>Ini adalah contoh website yang dibuat dengan HTML dan CSS.</p> </section> </main> <footer> <p>© 2024 Website Saya</p> </footer> </body> </html>

Kesimpulan

Dengan memahami dasar-dasar HTML dan CSS, Anda sudah memiliki bekal untuk membuat website yang menarik. Latih terus keterampilan ini, dan eksplorasi berbagai cara untuk mengkreasikan desain yang unik.
